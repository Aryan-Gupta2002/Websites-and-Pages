# 🚗 Sports Car Carousel

Interactive car showcase webpage with dynamic video backgrounds and carousel UI.

## 🔗 Deployed URL
https://mclaren-car-carousel.vercel.app/#

## ✨ Features
- Carousel for car selection
- Background video changes per model
- Dynamic model name update
- Play/Pause video control
- Responsive design

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Materialize CSS
- jQuery
